# Block hash calculator

Web app to visualize the calculation of a block's hash on Ethereum and EVM-like blockchains.

Code repository mirrors: [GitHub](https://github.com/ardislu/block-hash-calculator), [Codeberg](https://codeberg.org/ardislu/block-hash-calculator), [git.ardis.lu](https://git.ardis.lu/block-hash-calculator)

## Tests

Requires Node.js **v20+**.

1. Install dependencies:

```
npm install
```

2. Run the test suite:

```
npm test
```
